## 功能

| 編號 | 功能 | 描述 |
|---|---|---|
|1|友情鏈接|在```/public/friends.json```填入友情鏈接的信息后，就可以在 https://domain.com/#/friends 中查看友鏈信息|

## 目錄樹

```
└── assets/      # 樣式與圖片
public/
└── friends.json # 友鏈數據
```
